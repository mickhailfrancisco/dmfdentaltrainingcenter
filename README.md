## About

This repository powers the official web application for **DMF Dental Training Center** — a training provider for dental professionals in the Philippines.

The platform lets students browse programs, enroll online, and complete payments. It also gives the DMF team the tools to manage courses, track enrollments, verify payments, and run day-to-day operations.

This project is **owned by DMF Dental Training Center**. The software is proprietary and intended for authorized use only.
